# Car Dealship

Aplicación básica desarrollada en Nest JS. Principios básicos de una API Rest.

Ejecutar Aplicación en modo desarrollo
```
yarn start:dev
```

Construir y Ejecutar Aplicación en modo producción
```
yarn build
yarn start:prod
```

Populate Data
```
http://localhost:3000/seed
```
